Audit each component for image optimization per D-09:

1. Check if any component uses `<img>` tags - convert to `next/image`:
```tsx
// Bad
<img src="/logo.png" alt="Logo" />

// Good
import Image from "next/image";
<Image src="/logo.png" alt="Logo" width={200} height={100} />
```

2. For images above the fold (hero, header logo), add `priority`:
```tsx
<Image src="/logo.jpg" alt="..." width={...} height={...} priority />
```

3. For images below the fold (features, target-audience), ensure lazy loading (default):
```tsx
<Image src="/feature.svg" alt="..." width={...} height={...} loading="lazy" />
```

4. Add explicit width/height or use fill with sizes to prevent CLS:
```tsx
// Fixed size
<Image src="/icon.svg" alt="..." width={48} height={48} />

// Responsive fill
<div className="relative w-full h-64">
  <Image src="/bg.jpg" alt="..." fill sizes="100vw" className="object-cover" />
</div>
```

5. For SVG icons that are small and inline, can keep as `<img>` or import directly.

If no `<img>` tags are found (all using SVG inline or no images), document this finding.
  </action>

  </verify>
