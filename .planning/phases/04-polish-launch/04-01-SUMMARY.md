## Implementation Details

### lib/schema.ts
- `buildOrganizationSchema()`: Returns Organization JSON-LD with name, url, logo, description
- `buildWebSiteSchema()`: Returns WebSite JSON-LD with name, url, description, inLanguage
- Both typed with TypeScript interfaces

### app/robots.ts
- Allows all user agents on `/`
- Disallows `/api/` (no indexing of API routes)
- References sitemap at `https://generation-ai.org/sitemap.xml`

### app/sitemap.ts
- Returns root URL with priority 1, weekly change frequency
- Ready to extend when more pages are added

- **openGraph**: Complete with title, description, images array (1200x630)

- **alternates.canonical**: https://generation-ai.org
- **JSON-LD**: Two script tags injecting Organization and WebSite schemas
