  <action>
Create OG image per D-04 using ImageMagick (available on macOS).

Design approach (Claude's discretion per CONTEXT.md):
- Dark background (#141414) matching the website dark mode
- Generation AI logo centered
- Tagline below: "Die KI-Community fuer Studierende"
- Subtle gradient or accent color (#CEFF32) for visual interest

Steps:
1. Check ImageMagick availability: `which convert` or `which magick`
2. Create 1200x630 canvas with dark background
3. Composite the logo (resize from generationai-blau-neon.jpg to fit)
4. Add tagline text
5. Export as JPEG with quality 85 (balance quality vs size)
6. Verify file size < 500KB

If ImageMagick not available, use alternative approach:
- Use sharp via npx (Node.js image processing)
- Or create a simple branded image programmatically

Command structure (adjust based on tool availability):
```bash
# Using ImageMagick
magick -size 1200x630 xc:'#141414' \
  \( public/logos/generationai-blau-neon.jpg -resize 400x400 \) \
  -gravity center -composite \
  -font Helvetica -pointsize 36 -fill '#CEFF32' \
  -gravity south -annotate +0+60 'Die KI-Community fuer Studierende' \
  -quality 85 public/og-image.jpg
```

Alternative with sharp (if ImageMagick unavailable):
```bash
npx sharp-cli --input public/logos/generationai-blau-neon.jpg \
  --resize 1200 630 --background "#141414" \
  --output public/og-image.jpg
```
